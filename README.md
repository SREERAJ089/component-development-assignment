# Component Development Assignment
This project was developed as part of a technical assignment for an interview process. It implements a comment component with multiple UI states using React and Tailwind CSS. The component allows users to add comments, view existing comments in a dialog, and attach supporting documents.

## Features
- Comment form with validation
- File upload support
- Submitted comment display
- Scrollable comment dialog
- Dynamic comment list
- Clean UI built with Tailwind CSS

## Technologies
- React
- Tailwind CSS
- React Icons
- Vite
